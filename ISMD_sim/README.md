ISMD Simulator
Программный комплекс для макроскопического детерминированного моделирования пространственной динамики популяций на основе метода неоднородных пространственных моментов (Inhomogeneous Spatial Moment Dynamics).

В основе алгоритма лежит численное интегрирование интегро-дифференциальных уравнений с использованием замыкания Кирквуда. Для ускорения расчета нелокальных пространственных сверток применяются тензорные вычисления на GPU.

Для корректной работы симулятора необходимо наличие графического процессора (GPU) с поддержкой CUDA.

Зависимости:

Python 3.8+

NumPy

CuPy

SciPy

Matplotlib (для визуализации результатов)

Настройка параметров
Базовые демографические и пространственные параметры задаются непосредственно в коде (секция инициализации).
Параметры сетки:
L — размер расчетной одномерной области.
dx — шаг пространственной дискретизации.
dt — шаг интегрирования по времени.
Параметры популяции:
m — фоновая удельная смертность.
c_plus, c_minus — нормировочные коэффициенты (интенсивности) рождаемости и конкуренции.
sigma_plus, sigma_minus — эффективные радиусы разлета потомства и внутривидовой конкуренции.
Типы ядер (Kernels):Поддерживаются два типа ядер взаимодействия:
gaussian — нормальное распределение.
tophat — равномерное распределение с компактным носителем.
Результаты 
По завершении итераций симулятор:
Вычисляет асимптотическую скорость распространения популяционного фронта и выводит ее в консоль.
Генерирует и сохраняет графики пространственного распределения плотности $n_t(x)$ (профиль волны).
